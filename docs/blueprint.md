# **App Name**: helpmelearnthis

## Core Features:

-   Content Upload: Users can upload files of type PDF, DOCX, PPT, TXT, or paste plain text into a text area to serve as input or context for quizzes.
-   Theme Toggle: Implement a user-friendly interface to toggle between light and dark modes, maintaining the Vercel theme aesthetics for both modes.
-   Dynamic Question Generation: Use an LLM as a tool, providing quiz questions based on the content provided, tailoring questions based on topic relevance. Ensure questions are relevant to uploaded documents or text.
-   Difficulty Selection: Implement a difficulty adjustment feature, with user-selectable difficulty levels influencing question complexity.
-   Gamified Elements: Incorporate gamification elements such as points, badges, and leaderboards to enhance user engagement.
-   Adaptive Difficulty: Utilize AI to adjust quiz difficulty based on the user's performance. Difficulty changes only in the positive direction when the tool determines the user is ready.
-   Performance Feedback: Provide immediate feedback on answers and a comprehensive performance summary at the end of each quiz.

## Style Guidelines:

-   Primary color: Vercel Blue (#0076FF), used as the primary brand color of Vercel.
-   Background color: Light mode (#F0F4F8) provides a clean, accessible backdrop for content. Dark mode (#1A1A1A) for reduced eye strain in low-light environments.
-   Accent color: Light Gray (#CCD1D9), slightly different in saturation from background.
-   Font: 'Inter', a grotesque-style sans-serif (both headline and body).
-   Use simple, outlined icons from a consistent set (e.g., Font Awesome or Material Icons) for a clean and modern look. Use icons in the accent color.
-   Maintain a clean, intuitive layout with ample white space. Use a responsive design that adapts to different screen sizes, ensuring usability on both desktop and mobile devices.
-   Incorporate subtle animations and transitions to enhance user experience, such as smooth transitions between questions and feedback display. Ensure animations are performant.
